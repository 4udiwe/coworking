# Gateway
Принимает внешние HTTP запросы и проксирует их до нужного сервиса. Логирует метод, время ответа и статус.


## Rate Limit

[Rate_limit](internal/middleware/rate_limit.go) беспечивает контроль числа запросов в секунду от одного устройства. При пересечении границы RPS выдается ошибка `429 TooManyRequests`

## Routing

Вся настройка маршрутов для сервисов системы производится в [config.yaml](config/config.yaml), без редактирования кода.
